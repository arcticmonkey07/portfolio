Pepelane

Самостоятельный проект по nuxt js.

Сайт аренды летательных средств.

Возможности:

  • Загрузка итемов из json файла. Имитация загрузки с API.
  • Фильтрация.
  • Форма добавления нового транспортного средства.
  • Резиновая верстка, мобильная версия.
  • Темный режим.
  • Детальная страница итама с табами.
  • Стили написаны с помощью css-modules.
