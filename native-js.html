Candy-shop

    Candyshop. Проект показывающий изученные возможности JavaScript'a.

  • Загрузка товаров происходит с сервера с помощью AJAX.
  • Каждая карточка отрисовывается по шаблону.
  • Реализована фильтрация и сортировка товара.
  • В фильтрации цены реализованы слайдеры.
  • Можно отметить избранные товары, посмотреть состав товара, добавить в корзину.
  • Сверху отображается кол-во товаров в корзине.
  • Изменение товара в корзине по кол-ву, удаление из корзины.
  • Оформление заказа: валидация номера карты происходит по алгоритму Луна.
  • Проверка остальных полей карты реализована с помощью регулярных выражений.
  • Выбор доставки.
  • Отправка формы на сервер с помощью AJAX-запроса.
