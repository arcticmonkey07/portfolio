The Meals App

Учебный проект по react-native.

Приложение создано с помощью Expo.

Открыть его можно по QR-коду, скачав приложение Expo.

Описание

Мое второе учебное приложение на React Native.

Приложение с рецептами блюд.

Возможности:

  • Добавлена навигация: react-navigation-stack. Навигация внизу по табам: react-navigation-tabs. Навигация по меню-бургеру: react-navigation-drawer.

Guess Number App

Учебный проект по react-native.

Приложение создано с помощью Expo.

Открыть его можно по QR-коду, скачав приложение Expo.

Описание

Мое первое учебное приложение на React Native.

Игра Угадай Число. Нужно ввести любое число. На след. экране кнопками (+) и (-) направлять компьютер чтобы он угадывал число.

Возможности:

  • Кнопка 'Reset' удаляет введенные цифры в инпут.
  • На экране игры выводятся все числа которые угадывает компьютер.
  • В конце игры выводится число которое комрьютер угадывал.
  • В конце игры выводится сколько раундов понадобилось компьютеру, чтобы угадать число.
